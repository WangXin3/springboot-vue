# SpringBoot-Vue 后台管理系统

## 项目简介
一个基于 Spring Boot、Mybatis、JWT、Spring Security、Vue的前后端分离的后台管理系统

前端使用的是`PanJiaChen`大佬的开源vue-admin-template模板

地址：https://github.com/PanJiaChen/vue-admin-template
